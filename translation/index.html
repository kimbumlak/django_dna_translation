Nucleotide FASTA Sequence Translation

Paste one or more Nuleotide FASTA sequences into the text area below to get Protein Sequences:

(FASTA sequence file can be obtained from Genbank)

[Safa Khalid]. (2017, March 2). How to find FASTA Format from NCBI. Retrieved from https://www.youtube.com/watch?v=2TlIjPg_tIQ


Input Example in the text area:

>sequence1

ACGTACGTACGTACGTACGT

>sequence2

ACGUACGUACGUACGUACGU


WARNING! Sequence should only contain Nucleotide Bases (A,C,G,T or U)
